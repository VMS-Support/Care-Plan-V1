# Reference Rules

`RULE-CAREPLAN-COVERAGE-001` evaluates high-risk assessment coverage against active RLT-domain care-plan problems and recommends review only.

`RULE-WEIGHT-001` compares current and previous weights using configurable fixture thresholds; it remains pending clinical approval.

`RULE-CAREACTION-MISSED-001` counts missed care-action records in a configured lookback; it remains pending clinical approval.

`RULE-TEST-DEDUPE-001` is active and non-clinical. It proves idempotent event processing and deterministic output deduplication.
