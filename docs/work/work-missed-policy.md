# Work Missed Policy

Missed is an explicit terminal outcome, not an aged Overdue item. It requires reason code, time, actor/system policy, follow-up and escalation flags, audit, and any source event. Supported contract codes include resident refused/absent/in hospital, clinically inappropriate, staff/equipment unavailable, appointment not attended, expired window, unable to complete, and other. Automatic shift-end conversion is prohibited until clinically approved.
